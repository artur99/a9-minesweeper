# A9 MineSweeper
This is a game made as a challenge between <a href="https://github.com/artur99/a9-minesweeper">me</a>, <a href="http://github.com/pmaxim98">pmaxim98</a> and <a href="http://github.com/iulys01">iulys01</a>. :D

## About
Yet another Minesweeper game, but this one is made on a JS canvas.<br>
Thanks to <a href="kenney.nl/assets">Kenney.nl</a> for the graphics in the game, and to <a href="https://opengameart.org/content/game-backgrounds">OpenGameArt</a> for the background.<br>
Time been working on it: 3 days.

## Features
* 3 difficulty levels
* Right click flags
* Instructions
* Time Counter
* Highest Score

## Installation
Just clone the repo and run `game/index.html` in your browser.

## Development
For development, you need npm, bower & grunt :P <br>
```shell
npm install
bower update
grunt prep
grunt watch
```
And after stopping the watcher, for minifying & other things, simply run `grunt` again.
